# 🐍 贪吃蛇游戏

一个支持多人在线对战、云端数据同步的贪吃蛇游戏。

## 🎮 游戏特性

- **用户系统** - 注册、登录、成就系统
- **云端存储** - Firebase 实时数据库，游戏数据自动同步
- **多种模式** - 经典模式、限时挑战、无尽模式、双人对战
- **道具系统** - 加速、减速、无敌等多种道具
- **皮肤系统** - 自定义蛇身颜色、食物样式、背景
- **音效系统** - 可开关的背景音乐和游戏音效

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 暂停/继续 |
| F | 全屏切换 |

## 🛠️ 技术栈

- HTML5 Canvas
- JavaScript (原生)
- Firebase Realtime Database
- EmailJS 邮件服务

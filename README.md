# 贪吃蛇游戏 - 使用说明

## 游戏目标
控制蛇吃到食物，获得尽可能高的分数。

## 控制方式

### 键盘控制
| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 暂停/继续 |
| Enter | 开始游戏 |

### 游戏设置
- **游戏时间**：设置游戏时长 (0-300秒，0为无限)
- **游戏速度**：调整蛇的移动速度 (1-10级)

## 游戏规则
1. **计分规则**：蛇每次吃到食物得 `10 + 速度×2` 分
   - 速度 1 级：每次 12 分
   - 速度 5 级：每次 20 分
   - 速度 10 级：每次 30 分
2. 蛇撞到墙壁或自己的身体则游戏结束
3. 游戏结束后会自动保存记录

## 界面说明
- 顶部：玩家姓名输入
- 中部：游戏画布和设置
- 底部：游戏记录列表

## 常见问题
**Q: 如何开始游戏？**
A: 输入玩家姓名，点击"开始游戏"按钮或按 Enter 键。

**Q: 如何暂停游戏？**
A: 点击"暂停"按钮或按空格键。

**Q: 记录会自动保存吗？**
A: 是的，游戏结束后会自动保存到本地存储。

# 贪吃蛇游戏 - 使用说明

## 游戏目标
控制蛇吃到食物，获得尽可能高的分数。

## 控制方式

### 键盘控制（单人模式）
| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 暂停/继续 |
| F | 全屏切换 |

### 键盘控制（双人对战模式）
| 玩家 | 按键 | 功能 |
|------|------|------|
| P1 | W/A/S/D | 控制移动 |
| P2 | 方向键 | 控制移动 |

## 游戏模式

### 经典模式
- 传统贪吃蛇玩法
- 可设置游戏时间（0为无限）
- 可启用道具系统

### 限时挑战
- 60/90/120秒时限挑战
- 在限定时间内获得最高分
- 包含道具系统

### 无尽模式
- 没有时间限制
- 每吃5个食物速度+1
- 挑战极限高分

### 双人对战
- 双人同屏对战
- P1: WASD控制
- P2: 方向键控制
- 先撞墙者输

## 道具系统

### 食物类
| 道具 | 图标 | 效果 |
|------|------|------|
| 普通食物 | 🍎 | 得分 10 + 速度×2 |
| 金色食物 | ★ | +50分 |

### 障碍类
| 道具 | 图标 | 效果 |
|------|------|------|
| 炸弹 | 💣 | -20分，蛇身缩短2节 |

### 加速类（永久提升基础速度）
| 道具 | 图标 | 效果 |
|------|------|------|
| 汽车 | 🚗 | 速度 +1 |
| 高铁 | 🚄 | 速度 +2 |
| 飞机 | ✈️ | 速度 +3 |

### 减速类（永久降低基础速度）
| 道具 | 图标 | 效果 |
|------|------|------|
| 蜗牛 | 🐌 | 速度 -3 |
| 考拉 | 🐨 | 速度 -2 |
| 乌龟 | 🐢 | 速度 -1 |

### 特殊类
| 道具 | 图标 | 效果 |
|------|------|------|
| 无敌 | 🛡️ | 获得1次穿墙机会（撞墙不死） |
| 幽灵 | 👻 | 获得1次穿过自身机会（撞自己不死） |

**注意：** 加速/减速道具是永久改变基础速度，游戏重启后恢复。

## 计分规则
- 基础分数: 10 + 速度×2
- 速度 1 级：每次 12 分
- 速度 5 级：每次 20 分
- 速度 10 级：每次 30 分

## 常见问题

**Q: 如何开始游戏？**
A: 输入玩家姓名，选择游戏模式，点击"开始游戏"按钮或按空格键。

**Q: 如何暂停游戏？**
A: 点击"暂停"按钮或按空格键。

**Q: 如何进入双人对战？**
A: 在游戏模式中选择"双人对战"，点击开始游戏即可。

**Q: 记录会自动保存吗？**
A: 是的，游戏结束后会自动保存到本地存储。

**Q: 如何解锁成就？**
A: 点击"成就"按钮查看当前进度，完成相应条件即可解锁。解锁时屏幕会显示祝贺词。

**Q: 如何更换皮肤？**
A: 点击"皮肤"按钮，选择喜欢的样式后保存即可。

**Q: 如何调整音效？**
A: 点击"音效"按钮，可以单独开关各类音效。

**Q: 如何注册和登录？**
A: 打开游戏后会自动显示登录/注册界面。输入用户名和密码（至少3位）即可注册。注册后可选择绑定 GitHub Token 进行云同步。

**Q: 如何使用 GitHub 云同步？**
A: 1. 注册/登录后，在登录界面下方输入 GitHub Personal Access Token
   2. 创建 Token 方法：GitHub → Settings → Developer settings → Personal access tokens → Generate new token
   3. 勾选 gist 权限
   4. 点击"同步到 GitHub"按钮
   5. 您的游戏数据将保存到 GitHub Gist 中

**Q: 无尽模式怎么玩？**
A: 选择"无尽模式"，每吃5个食物速度会自动+1，没有时间限制，挑战最高分！

**Q: 道具效果是永久的吗？**
A: 加速/减速道具是永久改变本局基础速度，汽车+1高铁+2飞机+3，蜗牛-1考拉-2乌龟-3。无敌道具提供1次撞墙保护机会。
